# 🧠 ResumeIQ — AI Resume Analyzer

ResumeIQ is a full-stack web application that analyzes resumes (PDF/DOCX), detects skills and key sections, and generates a resume score with improvement suggestions in a modern dashboard.

---

## 🚀 Features
- Upload resumes in **PDF or DOCX**
- Extract and analyze **skills, sections, and content**
- Generate **resume score (0–100)**
- Show **missing skills and improvement suggestions**
- **Interactive dashboard** with charts and progress bars
- **Drag-and-drop upload**
- **Dark mode support**
- **Analysis history** saved locally
- Fully **responsive design**

---

## 🛠 Tech Stack
**Frontend:** React, Vite, CSS  
**Backend:** Node.js, Express  
**File Processing:** Multer, pdf-parse, mammoth
